项目开源免费，求上面点星支持(star ^o^)

本框架工程基于Unity 5.0 + UGUI + tolua构建
服务器端基于VS2012及其以上版本。

有问题请加：ulua技术交流群 434341400

支持平台：PC/MAC/Android(armv7-a + Intel x86)/iOS(armv7 + arm64)

**资料地址**<br>
